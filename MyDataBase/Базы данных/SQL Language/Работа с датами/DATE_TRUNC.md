Отрезки времени:

| microseconds | микросекунды |
| ------------ | ------------ |
| milliseconds | миллисекунды |
| second       | секунда      |
| minute       | минута       |
| hour         | час          |
| day          | день         |
| week         | неделя       |
| month        | месяц        |
| quarter      | квартал      |
| year         | год          |
| decade       | десятилетие  |
| century      | век          |

Таким образом передав нужный отрезок времени в эту функцию мы получим записи за нужное нам время.
