Чтобы запустить PowerShell скрипт, который вы создали для работы с Git, вам нужно выполнить несколько шагов. Вот как это можно сделать:

Установка Posh-Git: Для улучшения работы с Git в PowerShell рекомендуется установить модуль Posh-Git. Это можно сделать с помощью команды Install-Module posh-git -Scope CurrentUser -Force в PowerShell. Этот модуль добавляет расширенные возможности Git в вашу консоль PowerShell, включая автодополнение и информацию о состоянии репозитория в приглашении к вводу 3.
Настройка ExecutionPolicy: По умолчанию, PowerShell может блокировать выполнение скриптов из-за политики безопасности ExecutionPolicy. Чтобы разрешить выполнение вашего скрипта, вам нужно изменить эту политику. Вы можете установить политику RemoteSigned для текущего пользователя с помощью команды Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned -Force. Это позволит запускать скрипты, которые были созданы на вашем компьютере, а также скрипты, подписанные доверенным издателем 3.
Запуск скрипта: После установки Posh-Git и настройки политики ExecutionPolicy, вы можете запустить ваш скрипт. Откройте PowerShell, перейдите в директорию, где находится ваш скрипт, и выполните его, введя .\имя_вашего_скрипта.ps1.
